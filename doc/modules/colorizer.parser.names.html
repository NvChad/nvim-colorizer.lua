Module colorizer.parser.names

This module provides a parser that identifies named colors from a given line of text.

It supports standard color names and optional Tailwind CSS color names. The module creates a lookup table and Trie structure to efficiently match color names in text.

name_parser (line, i, opts)
Parses a line of text to identify named colors Uses a Trie structure to search for the longest prefix that matches a color name in `color_map`.

Parameters:

  • line string The line of text to parse for color names
  • i number The index in the line where parsing should start
  • opts table Options, including whether Tailwind colors are enabled

Returns:

  1. number|nil The length of the matched color name, or `nil` if no match
  2. string|nil The RGB hex value of the matched color, or `nil` if no match
