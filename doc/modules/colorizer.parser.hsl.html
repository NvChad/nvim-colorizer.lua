number:hsl_function_parser (line, i, opts)
Parse a CSS `hsl()` or `hsla()` color function and return an RGB hex value. This function handles parsing HSL and HSLA CSS color functions, including the optional alpha (opacity) value in HSLA.

For more info: https://developer.mozilla.org/en-US/docs/Web/CSS/color_value/hsl

Parameters:

  • line string The line of text to parse
  • i number The index in the line where parsing should start
  • opts table Options containing configuration values, including `prefix` for function prefix (e.g., "hsl" or "hsla")

Returns:

  1. number|nil The index in the line where the HSL/HSLA function ended, or `nil` if parsing failed
  2. string|nil The RGB hex value as a string, or `nil` if parsing failed
